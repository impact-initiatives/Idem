# Get question names from an XLSForm

Returns the values of the `name` column from the `survey` sheet,
excluding any rows where `name` is `NA` (such as `begin_group` /
`end_group` rows that carry no name).

## Usage

``` r
xlsform_questions(x, ...)

# Default S3 method
xlsform_questions(x, ...)

# S3 method for class 'xlsform'
xlsform_questions(x, ...)
```

## Arguments

- x:

  An `xlsform` object.

- ...:

  Ignored; present for S3 method compatibility.

## Value

A character vector of non-`NA` question names from the survey sheet.

## Details

The returned vector is used internally by
[`validate_question_names()`](https://impact-initiatives.github.io/idem/reference/validate_question_names.md)
to compare question inventories across two forms.

## See also

[`xlsform_list_names()`](https://impact-initiatives.github.io/idem/reference/xlsform_list_names.md)
for list names referenced in the survey;
[`xlsform_choices_list_names()`](https://impact-initiatives.github.io/idem/reference/xlsform_choices_list_names.md)
for list names defined in the choices sheet.
